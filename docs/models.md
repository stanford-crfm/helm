# Models

HELM includes many built-in models (text, vision-language, text-to-image, audio-language) from various providers. The model registry is maintained in the source code.

For the full list of models and API reference, build the documentation with MkDocs from the repository root (`pip install -e . && mkdocs serve`) or see the [model metadata registry]({{ site.repo_url }}/tree/main/src/helm/benchmark/model_metadata_registry.py).
