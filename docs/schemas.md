# Schemas

HELM uses several core data structures (e.g. `Scenario`, `Instance`, `Request`, `RequestResult`). They are defined in the source code under `helm.benchmark` and `helm.common`.

For the full API reference, build the documentation with MkDocs from the repository root or see the [source code]({{ site.repo_url }}/tree/main/src/helm).
